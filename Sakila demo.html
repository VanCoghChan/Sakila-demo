import datetime
rental.sort_values(by='customer_id', inplace=True)
lease_time = []
for i in range(len(rental)):
    try:
        d1 = datetime.datetime.strptime(rental.loc[i,:]['rental_date'],'%Y-%m-%d %H:%M:%S')
        d2 = datetime.datetime.strptime(rental.loc[i,:]['return_date'],'%Y-%m-%d %H:%M:%S')
        d3 = d2-d1
        rent_time = d3.days + round(d3.seconds/86400, 2)
        lease_time.append(rent_time)
    except:
#         print(i)
        rent_time = 3600
        lease_time.append(rent_time)

我们通过大致的分析决定将满足与期望做差的绝对值大于3倍方差的值（即 abs(value - u）> 3 * std），以及数值直接大于2000的作为异常值¶

异常值填补：观察上面的数据基本的统计信息，决定将75%分位~222.0（除去异常值之后的最大值）之间的随机数作为填补数据更换掉异常值¶

顾客分类方案实现¶

根据实际情况我猜测最终的消费指标的形式为各列特征线性相加（ind = w1*x1 + w2*x2 + ..... +wi*xi）¶

经过分析各项权重分配如下：lease_time*0.2 + frequency*0.25 + amount_sum*0.45 + active*0.1¶
